# Linux Device Driver Practice

This directory contains my personal practice and experimentation for the Linux Device Driver course.

## Contents

- **001hello_world**: A basic "Hello World" kernel module.
- **002psuedo_char_driver**: Placeholder for a pseudo character driver.

## Notes

- Build kernel modules using the provided `Makefile`.
- Use `ARCH=arm` and `CROSS_COMPILE=arm-linux-gnueabihf-` for BeagleBone Black development.
- Target kernel directory: `~/workspace/ldd/source/linux_bbb_4.14/`
